Encabezados

En html se manejan 6 distintos encabezados, que se manejan del más importante H1 hasta el menos importante H6,
los titulos son importante ya que el navegador los usa para ver como esta la estructura y el contenido de la pagina web.

Parrafos

Los parrafos siempre comienza en una nueca linea y los navegadores añaden automaticamebte un espacio en blanco
(un margen) antes y despues de un parrafo.
Para poder hacer un salto de linea existe la etiqueta br que no comienza un nuevo parrafo, y exite la etiqueta hr
de igual manera que br, hace un salto de pagina, pero este se usa para escribir otro parrafo nuevo.
Para el caso de los poemas exite la etiqueta "pre" ya que tiene una tipografia por defecto y conserva los espacios como saltos de linea.

Elementos de citas

Para citar algun texto, frase o parrafo existen las siguiente etiquetas

  • La etiqueta blockquote define una seccion que se cita de otra fuente, los navegadores suelen aplicar sangria a estos.

    Estoy convencido de que la mitad de lo que separa a los emprendedores exitosos de los que no triunfan es la perseverancia,
    Si vives cada día de tu vida como si fuera el último, algún día realmente tendrás razón.
    Cuando se innova, se corre el riesgo de cometer errores. Es mejor admitirlo rápidamente y continuar con otra innovación

  • La etiqueta "q" define una cita breve.

    Hanma Baki- sss

  • La etiqueta abbr define una abreviatura o un acronimo

    OMS fue fundada el 7 de abril 1948.

  • La etiqueta address define informacion de contacto del autor.

    Escrito por Brian E.
    Siguelo en:
    https://beacons.page/bemg_develop
    Toluca, México.

  • La etiqueta cite, define el tirulo de una obra creativa

    La moricion hecho por Miriam,2021

  • La etiqueta se utiliza para anular la direccion actualo del texto
